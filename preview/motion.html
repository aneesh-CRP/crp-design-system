Rules

Do

✓ Default to --ease-out + --dur-fast for any hover

✓ Use --dur-slow only for modals, sheets, drawers

✓ Animate transform and opacity — cheap, composited

✓ Respect prefers-reduced-motion — drop to opacity or instant

Don't

✗ No bounces, springs, or overshoot on UI — brand is reassuring

✗ No durations > 400ms except a single marketing hero reveal

✗ Don't animate width, height, top/left — janky

✗ No shimmer skeletons — use a spinner or a calm fade-in
