Layout rules — text expansion

  • Spanish runs 15–25% longer. Never fix button widths — use padding + min-width so labels grow.
  • Hero headlines: budget one extra line at every breakpoint. Test El futuro de la medicina at 64px before shipping.
  • Nav items may wrap — cap primary nav at 5 items on /es/.
  • Market Pro script: same one-moment-per-page rule; the accented script line is comienza contigo.

Type & tone

  • Same Area Normal scale — no size changes for Spanish.
  • Use tú (warm, direct) — never usted on marketing; matches the "you" rule in English.
  • Sentence case everywhere; Spanish never title-cases headings.
  • Legal/consent copy is translated by a certified translator, never machine-only. IRB-approved consent language may not be paraphrased.
