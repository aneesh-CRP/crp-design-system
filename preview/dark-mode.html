  --crp-orange: #FFB266;   /* was #FF9933 */
  --crp-navy:   #e5ecf5;   /* text color role flips to light */

  /* Semantic tokens lighten for legibility on dark */
  --success:    #34d399;
  --danger:     #f87171;
  --warning:    #fbbf24;

  /* Soft backgrounds become rgba tints of the accent */
  --blue-bg:    rgba(90,128,212,.15);
  --orange-bg:  rgba(255,178,102,.15);

  /* Shadows get heavier and cooler */
  --shadow-sm:  0 1px 3px rgba(0,0,0,.4);
  --shadow-lg:  0 4px 16px rgba(0,0,0,.5);
}
Rules

Where dark mode applies. Dashboard only ([data-theme="dark"] on <html>, user-toggled, persisted in localStorage). Marketing stays light — dark clinical-trial marketing reads as untrustworthy.

What shifts. Luminance, not hue. Navy stays navy (cool-blue family); orange stays orange (warm family). We lighten brand blue and orange on dark backgrounds so they pass contrast against the new surface.

What doesn't shift. Gradients, Market Pro script (still orange), the quatrefoil stencil (uses currentColor).
